Equality Operator

JavaScript has loose and strict equality operators.

Loose Equality (==)

  • == compares only the value
  • It performs type coercion automatically ⚠️ 
console.log(5 == "5");     // true  ✅ (string "5" becomes number 5)
console.log(true == 1);    // true
console.log(false == 0);   // true

Strict Equality (===) ✅ Recommended

  • === compares both value + type
  • No type coercion happens ✅ 
console.log(5 === "5");    // false ✅ different types
console.log(true === 1);   // false
console.log(false === 0);  // false

Not Equal Operators (!= vs !==)

console.log(5 != "5");     // false (because "5" coerces to 5)
console.log(5 !== "5");    // true  ✅ strict check

Best Practice ✅

  • Always prefer === and !== for clean and predictable code.
  • Avoid == unless you intentionally want coercion.
