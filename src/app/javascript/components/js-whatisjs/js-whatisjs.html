What is JS

Brief Overview

High level program

  • Every program that runs on a computer needs some hardware resources such as memory, CPU, mouse, keyboard to do its work.
  • Languages: JS & Python
  • We don’t need to manage resources manually.
  • These languages have abstractions that take the work away from us.
  • The program that takes memory management away from us is called Garbage Collection.
  • Inside the JS engine, garbage collection automatically removes old, unused objects from memory.

Translation

  • Computer la run aagura ella program kum edho oru hardware theva padum like memory, CPU, mouse, keyboard adhoda velaiya seiradhukaga.
  • Languages: JS & Python
  • Program run aaguradhuku theva padura ellathaum namba manual ah manage pana vendam.
  • Idhu mathri run aagura program ellam abstractions nu soluvanga — namba kitta velaiya kudukama adhuve pathukum.
  • Memory management ah adhuve pathukura program ah garbage-collection soluvanga.
  • JS engine palaya use aagadha objects ah adhuvave automatic ah computer memory la irundhu remove panikum.

Low level language

  • We need to manually manage these resources.
  • Example: Ask the computer for memory to create a new variable.

Translation

  • Program run aaguradhuku theva padura ellathaum nambalave dha manage pananum.
  • Example: Computer kita ketu oru variable create panradhuku memory create pana solanum.

Computer processor

  • It only understands zeroes and ones (0101001).
  • It is not possible to write code in 0 & 1.
  • We simply write human-readable JS code.
