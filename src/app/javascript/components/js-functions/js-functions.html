Functions

Functions are reusable blocks of code designed to perform a specific task.

What is a Function?

  • A function allows you to reuse logic without writing it again and again.
  • You can pass values into a function using parameters.
  • The function can return an output using return.

1) Function Declaration

function greet() {
  console.log("Hello Tom!");
}

greet();

2) Function with Parameters

function add(a, b) {
  console.log(a + b);
}

add(10, 5);

3) Function with Return

function multiply(x, y) {
  return x * y;
}

let result = multiply(5, 4);
console.log(result);

4) Function Expression

Functions can also be stored in variables. 

const sayHi = function () {
  console.log("Hi!");
};

sayHi();

5) Arrow Functions (ES6)

Arrow functions are shorter syntax for writing functions. 

// Normal function
function subtract(a, b) {
  return a - b;
}

// Arrow function
const subtractArrow = (a, b) => {
  return a - b;
};

// One-liner arrow function
const square = (n) => n * n;

6) Default Parameters

function welcome(name = "Guest") {
  console.log("Welcome " + name);
}

welcome();       // Guest
welcome("Sam");  // Sam

7) Rest Parameters

Rest operator ... collects multiple arguments into an array. 

function sum(...numbers) {
  let total = 0;

  for (let num of numbers) {
    total += num;
  }

  return total;
}

console.log(sum(1, 2, 3, 4));

8) Callback Function (Basic)

A callback is a function passed into another function as an argument. 

function greetUser(name, callback) {
  console.log("Hello " + name);
  callback();
}

greetUser("Tom", function () {
  console.log("Welcome to JavaScript!");
});
