Loops

Loops are used to execute a block of code repeatedly until a condition is met.

1) for Loop

Used when you know how many times you want to repeat the code. 

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

2) while Loop

Executes the block while the condition is true. 

let i = 1;

while (i <= 5) {
  console.log(i);
  i++;
}

3) do...while Loop

Runs the code at least once even if the condition is false. 

let j = 1;

do {
  console.log(j);
  j++;
} while (j <= 3);

4) break and continue

  • break → stops the loop completely
  • continue → skips current iteration
for (let i = 1; i <= 5; i++) {
  if (i === 3) continue; // skip 3
  if (i === 5) break;    // stop loop at 5
  console.log(i);
}

5) for...of Loop

Best for iterating over arrays. 

let colors = ["red", "green", "blue"];

for (let color of colors) {
  console.log(color);
}

6) for...in Loop

Best for iterating over object keys. 

let user = {
  name: "Tom",
  age: 24
};

for (let key in user) {
  console.log(key, user[key]);
}
