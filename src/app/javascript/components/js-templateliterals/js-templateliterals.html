Template Literals

Template literals make string creation easier using backticks and interpolation.

What are Template Literals?

  • Template literals use backticks: `...`
  • They support variable interpolation using ${variable}
  • They support multi-line strings easily âœ…

String Concatenation vs Template Literals

// Old way (concatenation)
const firstName = "Tom";
const job = "Developer";
const year = 2026;

const msg1 = "I'm " + firstName + ", a " + job + " in " + year;
console.log(msg1);

// Template literals (recommended)
const msg2 = `I'm ${firstName}, a ${job} in ${year}`;
console.log(msg2);

Expressions inside ${ }

You can write any valid expression inside ${ }. 

const a = 10;
const b = 20;

console.log(`Sum is ${a + b}`);         // 30
console.log(`Is a greater? ${a > b}`);  // false
