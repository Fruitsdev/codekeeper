Operators

Operators are symbols used to perform operations on values and variables.

1) Arithmetic Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (Remainder)
  • ** Exponentiation
  • ++ Increment
  • -- Decrement
let a = 10;
let b = 3;

console.log(a + b);  // 13
console.log(a - b);  // 7
console.log(a * b);  // 30
console.log(a / b);  // 3.333...
console.log(a % b);  // 1
console.log(a ** b); // 1000

a++;
b--;

2) Assignment Operators

  • = Assign
  • +=, -=, *=, /= Update and assign
let x = 10;

x += 5;  // x = 15
x -= 2;  // x = 13
x *= 2;  // x = 26
x /= 2;  // x = 13

3) Comparison Operators

  • == Equal (checks value only)
  • === Strict equal (checks value + type) ✅
  • != Not equal
  • !== Strict not equal ✅
  • >, <, >=, <=
console.log(5 == "5");   // true  ✅ value only
console.log(5 === "5");  // false ✅ type mismatch

console.log(10 > 5);     // true
console.log(10 <= 9);    // false

4) Logical Operators

  • && AND
  • || OR
  • ! NOT
let age = 20;
let hasId = true;

console.log(age >= 18 && hasId); // true

console.log(age < 18 || hasId);  // true

console.log(!hasId); // false

5) Ternary Operator

A short way to write an if...else. 

let marks = 45;

let result = marks >= 40 ? "Pass" : "Fail";
console.log(result);

6) String Operators

You can use + to concatenate strings. 

let firstName = "Tom";
let lastName = "G";

console.log(firstName + " " + lastName); // Tom G
