Variable

Variables are containers used to store data values in JavaScript.

What is a Variable?

  • A variable is like a container that holds a value.
  • The value can be changed later in the program.
  • Used to store numbers, strings, booleans, objects, arrays, etc.

Declaring Variables

In JavaScript, variables can be declared using var, let, and const. 

// using var (old way)
var name = "Tom";

// using let (recommended)
let age = 24;

// using const (constant value)
const country = "India";

Rules for Naming Variables

  • Must start with a letter, _, or $
  • Cannot start with a number
  • Cannot use reserved keywords (example: if, class)
  • Use meaningful names (example: userName, totalPrice)
  • JavaScript is case-sensitive (age and Age are different)

Difference: var vs let vs const

  • var: function scoped, can be re-declared (avoid using)
  • let: block scoped, can be re-assigned ✅
  • const: block scoped, cannot be re-assigned ✅
let score = 10;
score = 20; // ✅ allowed

const pi = 3.14;
// pi = 3.15; ❌ error
