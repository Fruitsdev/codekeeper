If else

if, else if, and else are used to execute code based on conditions.

1) Simple if

const age = 19;

if (age >= 18) {
  console.log("✅ You are an adult");
}

2) if...else

const age = 15;

if (age >= 18) {
  console.log("✅ You can vote");
} else {
  console.log("❌ You cannot vote");
}

3) else if Ladder

const marks = 82;

if (marks >= 90) {
  console.log("Grade A");
} else if (marks >= 75) {
  console.log("Grade B");
} else if (marks >= 50) {
  console.log("Grade C");
} else {
  console.log("Fail");
}

4) Multiple Conditions

const hasId = true;
const age = 20;

if (age >= 18 && hasId) {
  console.log("✅ Entry allowed");
} else {
  console.log("❌ Entry not allowed");
}

5) Nested if (Example)

const isLoggedIn = true;
const role = "admin";

if (isLoggedIn) {
  if (role === "admin") {
    console.log("✅ Welcome Admin");
  } else {
    console.log("✅ Welcome User");
  }
} else {
  console.log("❌ Please login");
}
