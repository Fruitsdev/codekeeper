Truthy & Falsy + Boolean Logic

Understand how JavaScript evaluates values in conditions and how logical operators work.

Truthy & Falsy

In JavaScript, values are converted to true or false automatically when used inside conditions.

Falsy values ✅

  • false
  • 0
  • "" (empty string)
  • null
  • undefined
  • NaN

Truthy values ✅

Everything else is truthy (example: "0", [], {}).

console.log(Boolean(0));       // false
console.log(Boolean(""));      // false
console.log(Boolean("Hello")); // true
console.log(Boolean([]));      // true
console.log(Boolean({}));      // true

Truthy/Falsy in if Condition

const money = 0;

if (money) {
  console.log("Don't spend it all!");
} else {
  console.log("You should get a job!"); // ✅ runs
}

Boolean Logic (AND / OR / NOT)

  • && AND → all conditions must be true
  • || OR → at least one condition must be true
  • ! NOT → reverses boolean value
const hasLicense = true;
const hasGoodVision = true;
const isTired = false;

console.log(hasLicense && hasGoodVision); // true
console.log(hasLicense || hasGoodVision); // true
console.log(!hasLicense);                 // false

Example: Can Drive?

const hasLicense = true;
const hasGoodVision = true;
const isTired = false;

if (hasLicense && hasGoodVision && !isTired) {
  console.log("✅ You can drive!");
} else {
  console.log("❌ Someone else should drive!");
}

Short-circuiting (Bonus ✅)

Logical operators don’t always return true/false. They return the actual value that made the result. 

// OR returns the first truthy value
console.log("" || "Tom");     // "Tom"

// AND returns the first falsy value
console.log(0 && "Hello");         // 0
