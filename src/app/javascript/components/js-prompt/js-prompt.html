prompt()

prompt() is used to get input from the user through a popup box.

What is prompt()?

  • prompt() displays a dialog box with an input field.
  • It returns the user input as a string.
  • If user clicks Cancel, it returns null. 
const name = prompt("What is your name?");
console.log(name);

Example: Convert input to Number ‚úÖ

Since prompt returns a string, we usually convert it using Number(). 

const ageInput = prompt("Enter your age:");
const age = Number(ageInput);

console.log(age);
console.log(typeof age);
