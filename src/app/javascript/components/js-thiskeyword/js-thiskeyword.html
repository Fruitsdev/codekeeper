this keyword

this depends on how a function is called.

Basics

  • If we set x = 5, then x will always be 5.
  • But the this keyword depends on the way in which a function is called.

4 different ways functions can be called

1) Method (Object)

  • Calling a function attached to an object is called a method.
  • When calling a method, this points to the object on which the method is called. 
const jonas = {
  userName: "Jonas",
  year: 1998,
  calcAge: function () {
    return 2023 - this.year;
  }
};

jonas.calcAge();
/*
calcAge - Method
this - jonas
year - 1998
*/

3) Arrow Function

  • Arrow functions do not get their own this.
  • They inherit this from the parent scope.

4) Event Listener

  • When a function is called as an event listener, this points to the DOM element that the handler is attached to.
