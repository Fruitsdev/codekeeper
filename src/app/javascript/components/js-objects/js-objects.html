Objects

Objects store data in key : value pairs.

What is an Object?

  • Objects represent real-world entities (example: user, product, car).
  • Data is stored using properties (keys) and values.
  • Objects can store strings, numbers, booleans, arrays, functions, etc.
let user = {
  name: "Tom",
  age: 24,
  isAdmin: false
};

Accessing Object Properties

  • Dot notation ✅ (most common)
  • Bracket notation ✅ (useful for dynamic keys)
let user = {
  name: "Tom",
  age: 24
};

console.log(user.name);     // Tom
console.log(user["age"]);   // 24

Updating / Adding Properties

let user = {
  name: "Tom"
};

user.age = 24;           // add
user.name = "Sam";       // update

user["country"] = "India";

console.log(user);

Deleting Properties

let user = {
  name: "Tom",
  age: 24
};

delete user.age;

console.log(user);

Object with Methods

An object can contain functions. These are called methods. 

let user = {
  name: "Tom",
  greet: function () {
    console.log("Hello " + this.name);
  }
};

user.greet();

Looping Through Object

let user = {
  name: "Tom",
  age: 24,
  country: "India"
};

for (let key in user) {
  console.log(key, user[key]);
}

Object.keys / values / entries

let user = {
  name: "Tom",
  age: 24
};

console.log(Object.keys(user));   // ["name", "age"]
console.log(Object.values(user)); // ["Tom", 24]
console.log(Object.entries(user));// [["name","Tom"], ["age",24]]
