Statements & Expressions

Understand the difference between expressions (produce values) and statements (perform actions).

Expressions ✅

An expression is a piece of code that produces a value.

  • Expressions can be stored in variables.
  • Expressions can be logged to console.
  • Expressions can be used inside template literals.
// Examples of expressions
3 + 4
1991
true && false
"Hello"

// expression can be stored
const age = 2026 - 2002;

console.log(age);

Statements ✅

A statement performs an action but does not produce a value directly. 

// if statement
if (age >= 18) {
  console.log("Adult");
}

// for loop statement
for (let i = 1; i <= 3; i++) {
  console.log(i);
}

Where Expressions are Used

const firstName = "Tom";
const job = "Developer";
const birthYear = 2002;

// Expressions inside template literals
const info = `I'm ${firstName}, a ${job}. Age: ${2026 - birthYear}`;
console.log(info);

Quick Summary

  • Expression → produces a value ✅
  • Statement → performs an action ✅
